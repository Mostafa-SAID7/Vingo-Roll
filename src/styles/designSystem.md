# Vingo Roll Design System

## Colors
- Primary Purple: `purple-600` (#9333EA)
- Secondary Pink: `pink-600` (#DB2777)
- Background: `from-purple-50 to-white`
- Text Primary: `gray-900`
- Text Secondary: `gray-600`
- Text Muted: `gray-500`

## Buttons
### Primary Button
```
bg-gradient-to-r from-purple-600 to-pink-600 text-white py-3 px-6 rounded-lg font-semibold hover:shadow-lg transition-all
```

### Secondary Button
```
border-2 border-purple-600 text-purple-600 py-3 px-6 rounded-lg font-semibold hover:bg-purple-50 transition-colors
```

## Cards
### Standard Card
```
bg-white rounded-2xl shadow-lg p-6 hover:shadow-xl transition-shadow
```

### Feature Card
```
bg-white rounded-2xl p-8 shadow-lg text-center hover:shadow-xl transition-shadow
```

## Spacing
- Page padding: `py-20`
- Container: `max-w-7xl mx-auto px-6`
- Section margin: `mb-12` or `mb-16`
- Element spacing: `space-y-6` or `gap-6`

## Typography
- Page Title: `text-5xl font-bold text-gray-900`
- Section Title: `text-3xl font-bold text-gray-900`
- Subtitle: `text-xl text-gray-600`
- Body: `text-gray-700`
